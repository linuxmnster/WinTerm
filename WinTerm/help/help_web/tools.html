🔍 Custom Tools

OSINT (Open-Source Intelligence) & Security Tools help investigators, cybersecurity professionals, and researchers **gather, analyze, and extract critical information from various sources. These tools are essential for penetration testing, ethical hacking, forensic analysis, and cybersecurity audits.

🛠 Available Security Tools

🔎 Binwalk - Firmware Analysis & Reverse Engineering

Description: Binwalk is a forensic analysis tool used for analyzing and extracting embedded files from firmware images, compressed archives, and hidden payloads. It is commonly used in **reverse engineering, IoT security, and malware analysis.

Features:

  • Extracts hidden files within firmware and binary images.
  • Identifies **compressed files, encryption, and executable code.**
  • Used for malware analysis and **security research.

Usage:

binwalk

📷 ExifTool - Metadata Extraction

Description: ExifTool is a powerful metadata extraction tool that analyzes images, PDFs, videos, and other digital files to uncover hidden data such as timestamps, geolocation, and device details.

Features:

  • Extracts metadata like camera details, GPS location, timestamps.
  • Analyzes forensic evidence in digital crime investigations.
  • Removes metadata to enhance privacy and security.

Usage:

exiftool

🌍 Google Search (GSearch) - OSINT Query Automation

Description: GSearch automates Google searches to find hidden or indexed data, making it a powerful OSINT tool for penetration testing, research, and competitive intelligence.

Features:

  • Automates Google dorking for advanced searches.
  • Extracts publicly available sensitive information.
  • Useful for threat intelligence & reconnaissance.

Usage:

google "site:example.com filetype:pdf" -number

🔎 Sherlock - Social Media Username Search

Features:

  • Finds usernames across social media, forums, and platforms.
  • Used for law enforcement, cybersecurity, and investigations.

Usage:

sherlock

🖼️ StegHide - Steganography & Hidden Data Extraction

Description: StegHide is a steganography tool that allows users to hide and extract secret data within image and audio files**. It is widely used in secure communication, digital forensics, and CTF (Capture The Flag) challenges**.

Features:

  • Hides secret messages or files inside images and audio.
  • Encrypts hidden data using password protection for security.
  • Extracts hidden data from suspicious or unknown images.
  • Used in forensics, cybersecurity, and covert communication.

Real-World Applications:

  • Used by journalists and activists to hide sensitive information from censorship.
  • Cybersecurity professionals use it for forensic analysis and ethical hacking.
  • Commonly seen in CTF competitions where hidden clues are embedded in images.

Usage:

                # To hide a secret file inside an image:
                steghide embed -cf cover.jpg -ef secret.txt -sf output.jpg -p mypassword
                
                # To extract a hidden file from an image:
                steghide extract -sf output.jpg -p mypassword
