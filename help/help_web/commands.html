⌨️ Terminal Commands

Terminal Commands are powerful text-based instructions that allow users to interact with the operating system directly. These commands provide efficient ways to navigate, manipulate files, and manage system resources** without a graphical interface.

🛠 Available Terminal Commands

📄 cat - Display File Contents

Description: Prints the content of a file to the terminal.

Usage:

cat <file_name>

Example:

cat myfile.txt

📁 cd - Change Directory

Description: Moves the user into the specified directory.

  • Use 'cd ..' to move up one level.
  • Use 'cd -' to get back to Desktop.
  • Use 'cd /' to jump to root (C:/).

Usage:

cd <directory_path>

Example:

cd Documents/

🧹 clear - Clear Terminal Screen

Description: Clears all previous command output from the terminal.

Usage:

clear

Example:

clear

📋 cp - Copy Files and Directories

Description: Copies a file or directory from source to destination.

  • Use 'cp <source>, <destination>' to copy directories.

Usage:

cp <source> <destination>

Example:

cp file.txt backup.txt

💾 df - Display Disk Space Usage

Description: Shows the available and used disk space for mounted filesystems.

Usage:

df

Example:

df

📊 du - Show Disk Usage

Description: Displays the size of a file or directory.

  • Use 'du -h' for human-readable format.

Usage:

du [options] <file/directory>

Example:

du -h myfolder

👀 head - Display First Lines

Description: Shows the first 10 lines of a file by default.

Usage:

head <file_name>

Example:

head myfile.txt

🛑 kill - Terminate a Process

Description: Sends a termination signal to the process with the given Process ID (PID).

  • Use 'kill -f <PID>' for force termination.

Usage:

kill <PID>

Example:

kill 1234

📋 ls - List Directory Contents

Description: Lists files and directories in the current location.

Options:

  • -a Show hidden files

Usage:

ls [OPTION]... [FILE]...

Example:

ls -a

📁 mkdir - Create a New Directory

Description: Creates a new directory with the specified name.

  • -m for multiple directories

Usage:

mkdir <directory_name>

Example:

mkdir myfolder

🔄 mv - Move or Rename Files

Description: Moves a file or directory to a new location. Can also be used to rename files.

Usage:

mv <source> <destination>

Example:

mv oldname.txt newname.txt
mv file.txt Documents/

🌐 ping - Test Network Connectivity

Description: Sends ICMP packets to check if a host is reachable.

Usage:

ping <hostname/IP>

Example:

ping google.com

⚙️ ps - Display Active Processes

Description: Lists currently running processes.

Usage:

ps

Example:

ps

🧭 pwd - Print Working Directory

Description: Displays the absolute path of the directory the user is currently in.

Usage:

pwd

Example:

pwd
Output: C:\Users\Kira\Documents

🔄 restart - Restart the System

Description: Reboots the system immediately.

Usage:

restart [time_in_sec]

Example:

restart

🗑️ rm - Remove Files or Directories

Description: Deletes the specified file.

  • Use 'rm -f <directory>' to remove a file with its content.
  • Use 'rm -r <dir1>, <dir2>' to remove multiple files

Usage:

rm <file/directory_name>

Example:

rm myfile.txt
rm -r myfolder

🗂️ rmdir - Remove Empty Directory

Description: Deletes an empty directory.

  • -m to remove multiple directory.
  • -f to remove non empty directory.

Usage:

rmdir <directory_name>

Example:

rmdir myfolder

⏻ shutdown - Power Off the System

Description: Schedules or immediately shuts down the system.

  • Use 'shutdown' to shut down immediately.

Usage:

shutdown [time_in_sec]

Example:

shutdown

👁️ tail - Display Last Lines

Description: Shows the last 10 lines of a file by default.

Usage:

tail <file_name>

Example:

tail myfile.txt

✏️ touch - Create Empty File

Description: Creates a new empty file if it does not exist.

  • Use 'touch -m <file1>, <file2>' to create multiple files at a time.

Usage:

touch <file_name>

Example:

touch myfile.txt

🌳 tree - Display Directory Structure

Description: Shows the directory structure in a hierarchical tree-like format.

Usage:

tree [path]

Example:

tree

📦 unzip - Extract ZIP Contents

Description: Extracts the contents of a .zip archive.

Usage:

unzip <zip_file>

Example:

unzip archive.zip

🗜️ zip - Compress Files

Description: Creates a .zip archive containing the specified files or directories.

Usage:

zip <zip_file> <file/directory>

Example:

zip myarchive.zip file1.txt file2.txt
